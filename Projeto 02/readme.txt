Segundo Projeto (Academia FitnessAndStrong)
O objetivo desse projeto foi o de passar um layout criado no figma para uma página HTML.

O objetivo pessoal foi o aprendizado de conceitos básicos de HTML e CSS na prática. Dessa vez, adicionando mais conteúdo.

Conceitos como:
Tags semanticas;
Tags do tipo block ou do tipo line;
CSS Box-model;
Ferramenta de desenvolvimento DevTools no navegador para auxiliar;
Criaçao de background;
Como utilizar uma lista;
Criaçao de um botao;
